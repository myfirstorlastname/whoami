This page documents my progression through the Jr Penetration Tester learning path on TryHackMe. It's a practical foundation for aspiring ethical hackers, and one of the best curated paths to transition toward the OSCP and real-world offensive roles.

Note: The first module is complete (✅), others are work-in-progress 🕐.

  • Documentation is part of my learning loop.

🚀 Modules

🌐 Intro to Web Hacking ✅

Hands-on entry into web exploitation: enumeration, bypass, automation. A perfect first step for aspiring ethical hackers.

🧰 Burp Suite ✅

Learn to intercept, inspect, and modify HTTP requests with Burp Suite. Essential tooling for bug bounty and pentesting.

🌐 Network Security 🕐

Learn scanning, enumeration, and how attackers map and exploit networks. Core knowledge before system-level exploitation.

🔎 Vulnerability Research 🕐

Understand CVEs, exploit paths, and real-world impact. Analyze services and identify weak surfaces.

💣 Metasploit 🕐

Master exploitation lifecycle using Metasploit: scanning, gaining access, privilege escalation, and persistence.

🔐 Privilege Escalation 🕐

Escalate from user to root/system using enumeration and misconfigurations. Core skill for OSCP and red teaming.

🎓 JrPT Certification 🕐

Final exam: a simulated environment to validate your knowledge across the full path.

🎯 Purpose

  • ✔ Consolidate foundational knowledge before OSCP prep
  • ✔ Track progression with hands-on challenges
  • ✔ Share a transparent, reproducible learning journey

This path isn’t about ticking boxes—it’s about sharpening thinking, building fluency, and solving like an attacker.
