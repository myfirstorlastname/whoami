🎭 Red Teaming (TryHackMe : Learning Path)

"Red teaming is not about exploits—it’s about simulating real adversaries to test resilience."

🧠 About this Learning Path

The Red Teaming path on TryHackMe is designed to introduce offensive operators to the full cycle of adversary simulation. Unlike traditional penetration testing, red teaming emphasizes stealth, persistence, and realism. The modules progress from fundamentals and initial access, through post-compromise actions, to advanced evasion and Active Directory compromise.

📘 Modules

🎭 Red Team Fundamentals ✅

Introduction to red team mindset, engagements, threat intel, OPSEC, and C2 basics.

🚪 Initial Access ✅

Phishing, web exploits, and techniques to gain a first foothold in the target network.

🖥️ Post Compromise

Actions after initial breach: privilege escalation, credential dumping, persistence.

🕶️ Host Evasions

Techniques to bypass AV/EDR, obfuscate payloads, and remain undetected on endpoints.

🌐 Network Security Evasion

Evading firewalls, proxies, IDS/IPS, and blending C2 traffic with normal activity.

🔐 Compromising Active Directory ✅

Attacking Kerberos, abusing misconfigurations, and full-domain compromise scenarios.
Red Teaming is a discipline: simulate real adversaries, adapt to defenses, and aim for persistence.
