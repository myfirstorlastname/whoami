Web Fundamentals

Overview

I have successfully completed the Web Fundamentals learning path on TryHackMe. This path offers a solid introduction to how the web works and how to begin hacking web applications. It ends with a final module that validates essential concepts before moving on to more advanced topics.

The official certification is displayed on my LinkedIn profile. Below is a breakdown of the modules included.

Modules

🌐 How the Web Works

Understand core web components, HTTP requests, DNS, web servers, and how browsers communicate.

🕷️ Introduction to Web Hacking

Start exploring vulnerabilities like XSS, IDOR, file inclusion, and basic recon techniques.

🧪 Burp Suite

Learn how to intercept, modify, and replay HTTP requests using Burp Suite as a proxy tool.

📚 Web Hacking Fundamentals

Validate your knowledge through practical exercises on login bypass, parameter tampering, and more.
