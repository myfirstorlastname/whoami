Browsed is a medium difficulty Linux machine exposing a web application that processes user-supplied archives.

Improper handling of uploaded ZIP files results in arbitrary Chrome extension execution.

This allows bypassing localhost trust boundaries and leads to remote code execution as a local user.

A misconfigured sudo Python script is then abused to escalate privileges to root.

Scan the target with nmap.

Open ports discovered: 22, 80.

User - RCE via Untrusted Chrome Extension & Localhost Trust Bypass

Untrusted File Upload: The web application extracts attacker-controlled ZIP files and loads them as Chrome extensions.

Client-Side Trust Abuse: The extension executes JavaScript across virtual hosts and accesses services bound to localhost.

Stored XSS: An internal Markdown application renders unsanitized user input as HTML.

Command Injection: A vulnerable routine endpoint allows arbitrary command execution.

Impact: Remote code execution as user larry.

Weakest point: Executing Chrome on attacker-supplied ZIP archives.

Root - Privilege Escalation via Python Bytecode Cache Poisoning

Credential Access: SSH private keys are recovered from the compromised user account.

Sudo Misconfiguration: The user can execute a Python script as root without a password.

Insecure Python Import: The script imports a local module using cached bytecode.

Bytecode Poisoning: A world-writable __pycache__ directory allows execution of attacker-controlled code as root.

Impact: Full system compromise.

Weakest point: World-writable Python bytecode cache.
