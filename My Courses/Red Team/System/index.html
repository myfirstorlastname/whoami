System and Endpoint Security refers to the protection of individual devicesâ€”servers, workstations, laptopsâ€”from malicious access and exploitation. While system security focuses on the configuration and hardening of operating systems, endpoint security adds defensive measures like anti-malware, behavior monitoring, and isolation.

System Security includes secure configuration of OS-level components, managing permissions, and applying regular updates.
Endpoint Security covers strategies like EDR (Endpoint Detection and Response), privilege restriction, and forensic readiness in case of compromise.

From the Red Team perspective, a common exercise is the boot2root challenge: compromising a target machine from initial access (boot) to full administrative control (root). These exercises simulate real-world post-exploitation, focusing on lateral movement and privilege escalation.

The objective is to minimize the attack surface while ensuring visibility and control at the edge of your infrastructure.
