Cloud

Security in the cloud starts with understanding what's beneath it.

Introduction

Cloud Computing refers to the on-demand delivery of computing resources, servers, databases, storage, networks, over the Internet. It relies on providers like Azure, AWS, or Google Cloud.

The term infrastructure includes all physical and virtual components (servers, networks, firewalls, VMs, etc.) required to run cloud and on-prem services. It forms the technical backbone of any organization.

The Role of SOC Level 4

In a SOC (Security Operations Center) context, a Level 4 analyst (SOC N4) is typically involved in:

  • Incident response across hybrid environments (cloud/on-prem).
  • Monitoring cloud configuration, logging, and IAM policies.
  • Investigating compromises on platforms like Azure AD.

A solid understanding of both infrastructure and cloud architecture is essential to detect, correlate, and respond effectively to threats in distributed environments.
