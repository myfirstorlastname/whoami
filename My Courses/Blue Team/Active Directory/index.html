🛡️ Active Directory Hardening

Defending the kingdom means securing its keys, gates, and messengers.

🧠 About This Course

This course introduces core defensive concepts related to Active Directory (AD) security. AD is the backbone of identity and access management in Windows-based enterprise environments, protecting it is a critical duty of any Blue Team.

You'll learn how to harden configurations, monitor for suspicious activity, and respond to common attacks like Kerberoasting, DC syncs, and credential theft.

🎯 Key Objectives

  • Understand the AD architecture and common attack surfaces
  • Apply group policy settings for enhanced security
  • Monitor key AD logs and events
  • Detect known attack techniques (e.g., DCSync, pass-the-hash)
  • Respond to suspicious behavior in hybrid and on-prem environments

🔐 Course Sections

🛠️ AD Security Baselines

Understand Microsoft's security baselines, default trusts, and tiering models.

📘 Hardening Practices

Apply secure GPOs, enforce admin tiering, and reduce attack surface.

📊 Monitoring & Detection

Use Windows logs, Sysmon, and Defender for Identity to detect malicious activity.

📂 MITRE ATT&CK Techniques

Map threats to real-world techniques like credential dumping, DCSync, and privilege escalation.

⚔️ Key Blue Team Skills

🔍 Monitoring Domain Events

Track logins, account lockouts, SID history changes, and LDAP queries.

🧱 Group Policy Management

Lock down AD with secure GPOs and enforce least privilege access.

🔐 Privileged Access Management

Control admin rights using tiered access, LAPS, and just-in-time elevation.

📉 Attack Surface Reduction

Disable unnecessary services, clean up stale accounts, and review trusts.

🛡️ Detection Engineering

Create alerts for common AD attacks and test them using Red Team simulations.

A secure Active Directory isn’t built in silence, it’s maintained by vigilance, knowledge, and continuous review.
